Destroyable wall ingredient used in the entire game, simulation done in Houdini, implemented in Snowdrop using the VAT workflow.
The dust and small particles were made by our VFX Artist - Baptiste Erades.
This was particulary challenging because it was the first time I was using Houdini in a production environment, took me a while to learn all the tips and tricks to better achieve this effect.
