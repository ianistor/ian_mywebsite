# Wooden Bridge

Animation is done using a mix of depth buffers, vertex shader and vertex positions baked on vertex color.
If needed vertex color can be deleted for planks that we do not want to move, however it wasn't the case here.
This is based on knowledge shared by Andrew Maximov at a GDC talk into Tech Art work at Naught Dog
